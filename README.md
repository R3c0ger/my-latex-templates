# Personal $\LaTeX$ Template Repository

<center> English | <a href="./README_zh-CN.md">中文</a> </center>

This repository contains some $\LaTeX$ templates that I have modified or written from scratch, provided for reference and use by everyone.

## Assignment

[Assignment-English](asgn-tpl/asgn-tpl.pdf): A template suitable for writing undergraduate or postgraduate course assignments, includes commonly used commands and formatting settings.

Referenced Projects:
- [HKUST Assignment Template by ZHAO, Yu Bo](https://www.overleaf.com/latex/templates/hkust-assignment-template/jvppndzfhdrj)

## Cheatsheet

[Cheatsheet-English](cheatsheet-tpl/cheatsheet-tpl.pdf): A template for creating course cheatsheets, formatted in A4 landscape layout, designed to be compact and easy to carry and consult.

Referenced Projects:
- [latex-cheatsheet by kenfehling](https://github.com/kenfehling/latex-cheatsheet)
- [cheatsheet-template by inyono](https://github.com/inyono/cheatsheet-template)

## Resume

[Resume-Chinese](resume-zh-cn-tpl/resume_zh_tpl.pdf): A template for creating personal resumes, uses the Microsoft YaHei font, features a highly compact layout with great flexibility.

Referenced Projects:
- [resume-ng by fky2015](https://github.com/fky2015/resume-ng)
