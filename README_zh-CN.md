# 个人 $\LaTeX$ 模板仓库

<center> 中文 | <a href="./README.md">English</a> </center>

这里存放着我个人魔改或编写的一些 $\LaTeX$ 模板，供大家参考和使用。

## 作业

[作业-英文](asgn-tpl/asgn-tpl.pdf)：适用于撰写大学本科或研究生课程作业的模板，包含常用命令和格式设置。

参考项目：
- [HKUST Assignment Template by ZHAO, Yu Bo](https://www.overleaf.com/latex/templates/hkust-assignment-template/jvppndzfhdrj)

## 小抄

[小抄-英文](cheatsheet-tpl/cheatsheet-tpl.pdf)：适用于制作课程小抄的模板，A4 横版版式，设计紧凑，便于携带和查阅。

参考项目：
- [latex-cheatsheet by kenfehling](https://github.com/kenfehling/latex-cheatsheet)
- [cheatsheet-template by inyono](https://github.com/inyono/cheatsheet-template)

## 简历

[简历-中文](resume-zh-cn-tpl/resume_zh_tpl.pdf)：适用于制作个人简历的模板，使用微软雅黑字体，布局高度紧凑，自由度较高。

参考项目：
- [resume-ng by fky2015](https://github.com/fky2015/resume-ng)
